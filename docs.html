PseudoPlay allows you to write and execute pseudocode in a simple, intuitive environment. All programs must start with START (or BEGIN) and end with STOP (or END).

Comments help explain your code and are ignored during execution. Use //, ', #, or -- for single-line comments, or /* ... */ for multi-line comments.

START

    // Single-line comment
    # Another single-line
    -- Yet another
    
    /*
      Multi-line
      comment
    */
    
    PRINT "Hello, World!"
    
STOP
Tip: Use comments liberally to make your code readable. Press Ctrl+Enter to run your code quickly.

Variables store data like numbers, strings, or booleans. Declare them with DECLARE (initializes to null) or INIT (declare and assign). Data types are inferred automatically.

Declaring Variables

START
    DECLARE name  // null by default
    INIT age = 25
    INIT message = "Hello"
    INIT isActive = TRUE
    
    // Multiple declarations
    DECLARE x, y, z
    INIT a = 1, b = 2
STOP

Assigning Values

Use SET, LET, or direct assignment to change values. Variables must be declared first.

START
    INIT score = 0
    SET score = 10
    LET score = score + 5
    score = 20  // Direct assignment
    
    PRINT score  // 20
STOP
Warning: Using undeclared variables will throw an error. Always declare before use.

Output with PRINT / OUTPUT

Display text, variables, or expressions. Combine with + or commas.

START
    INIT name = "User"
    PRINT "Hello, " + name + "!"
    OUTPUT "Score: ", 100
STOP

Input with INPUT / READ

Read user input and store in a declared variable. Optional prompt.

START
    DECLARE age
    PRINT "Enter your age:"
    INPUT age
    PRINT "You are " + age + " years old."
STOP
Tip: Input is treated as string unless numeric. Use expressions to convert if needed.

Use IF for decisions. Supports ELSEIF and ELSE. End with ENDIF.

START
    DECLARE score
    INPUT "Enter score:", score
    
    IF score >= 90 THEN
        PRINT "A Grade"
    ELSEIF score >= 70 THEN
        PRINT "B Grade"
    ELSE
        PRINT "Needs Improvement"
    ENDIF
STOP
Tip: Nest IF statements for complex logic, but keep indentation consistent.

FOR Loop

Repeat a fixed number of times. Optional STEP. End with NEXT.

START
    INIT i
    FOR i = 1 TO 5 STEP 1
        PRINT i
    NEXT
STOP

WHILE Loop

Repeat while condition is true. End with ENDWHILE.

START
    INIT count = 0
    WHILE count < 5
        PRINT count
        count = count + 1
    ENDWHILE
STOP

REPEAT UNTIL Loop

Repeat until condition is true (runs at least once).

START
    INIT num = 0
    REPEAT
        PRINT "Enter >10:"
        INPUT num
    UNTIL num > 10
STOP
Warning: Avoid infinite loops! There's a max iteration limit to prevent crashes.

Procedures and functions are defined but not yet executable. They are skipped during runtime.

PROCEDURE Greet(name)
    PRINT "Hello, " + name
ENDPROCEDURE
Note: Calling procedures/functions will error. Full support coming in future updates.
  • Use consistent indentation (auto-indent helps!).
  • Declare all variables at the top of your program.
  • Add comments for complex logic.
  • Test small parts of code incrementally.
  • Avoid deep nesting; use functions when available.
  • Undeclared Variable: Always use DECLARE/INIT first.
  • Mismatched Blocks: Ensure IF/ENDIF, loops match.
  • Infinite Loop: Check conditions; max iterations prevent hangs.
  • Syntax Error: Check quotes, operators, and keywords.
Tip: Use the line highlight to debug step-by-step.
